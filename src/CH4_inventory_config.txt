Config file utilized by main

Description:

     This function provides the main with all emission factors and
     similar user-edited input data as well as options for which
     sectors to run and, when applicable, which variations to run.

Usage:

     main_config()
     
Details:

     This function is intended to be the location for all user edited
     variables other than inputs to the main `CH4_inventory_build`
     function.

Value:

     Nothing is returned from the function.  It is run in
     `CH4_inventory_build` so that all user-edited settings are
     utilized throughout.
